# bilibiliStars
bilibili收藏夹快照

## 背景

![image-20230326193619585](assets/image-20230326193619585.png)

你是否也曾遇到过b站收藏的视频莫名其妙失效或被up主删除的情况。

由于b站的规则，你无法知道是哪位up主的哪一个视频被删除。

本项目旨在开发一个应用程序对您的b站收藏夹内视频的标题与封面进行备份操作，对已失效的视频进行标记等操作，来帮助您在这些视频失效时，了解到其内容。
